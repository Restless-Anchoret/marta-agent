# Marta Agent

Marta is an AI-powered personal assistant agent built on top of the Claude REST API, with a custom orchestration layer capable of handling various personal assistance tasks.

## Structure

- `backend/` — Kotlin/Spring REST API, orchestration layer, Claude API integration
- `frontend/` — TypeScript/React web interface

## Deployment

- Backend: [Fly.io](https://fly.io)
- Frontend: [Vercel](https://vercel.com)
